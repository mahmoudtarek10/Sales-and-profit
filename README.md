📊 Sales and Profit Dashboard Project

This project is a comprehensive Power BI dashboard designed to analyze sales performance and profitability across different dimensions. It provides clear, actionable insights into revenue, profit, customer behavior, and product performance.

The project demonstrates a complete end-to-end data workflow, making it ideal for learning Power BI, data modeling, and business analytics.

🔄 Project Workflow

The repository covers all stages from raw data to final insights:

📁 Raw Data
Original datasets before processing
Format: CSV / Excel
Includes Sales transactions, Products, Customers, and Regions data
🧹 Cleaned Data
Data after preprocessing and transformation
Handles:
Missing values
Duplicates
Data inconsistencies
Structured and ready for analysis
🧠 Data Modeling
Power BI data model (.pbix)
Relationships between tables (Fact & Dimension tables)
Implemented Star Schema for better performance

Key Measures include:

Total Sales
Total Profit
Profit Margin
Quantity Sold
Orders Count
📈 Dashboard / Visualizations

Interactive dashboard with dynamic visuals and KPIs:

Key Visuals:

KPI Cards:
Total Sales
